  30. def init (dbName: String): Unit

    Initialise a named MongoDB database

    Initialise a named MongoDB database

    This checks to see if the connection is to a http://cloudfoundry.org|CloudFoundry instance or not. If the connection is local such as in an offline installation or for testing then will connect to the database called dbName otherwise it will connect to the CF database specified in the connection settings that are provided by CloudFoundry

    Furthermore, a local connection assumes that MongoDB is running on port 27017 and is accessible via 127.0.0.1

    This will use the DefaultMongoIdentifier to identify the named database, so (I think) only one database can be used at a time.

    dbName

    The name of the local database to connect to. If connecting to a CloudFoundry database then this parameters is ignored and can be empty.
