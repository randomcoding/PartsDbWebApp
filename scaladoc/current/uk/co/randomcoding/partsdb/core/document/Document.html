uk.co.randomcoding.partsdb.core.document

Document

class Document extends MongoRecord[Document] with ObjectIdPk[Document]

Documents are the basic objects that make up a Transaction.

Documents basically contain a number of LineItems and a uk.co.randomcoding.partsdb.core.document.DocumentType.

Documents can additionally be editable. This indicates that the Document has not yet been completed, which in turn indicates the current stage of the Transaction.
